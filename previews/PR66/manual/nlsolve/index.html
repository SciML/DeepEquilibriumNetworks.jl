NonLinear Solvers

We provide the following NonLinear Solvers for DEQs. These are compatible with GPUs.

Note

If you are looking for general purpose nonlinear solvers, we recommend checking out NonlinearSolve.jl

DeepEquilibriumNetworks.BroydenSolver — Type
BroydenSolver()

Broyden Solver (Charles G Broyden (1965)) for solving Discrete DEQs. It is recommended to use LimitedMemoryBroydenSolver for better performance.

See also: LimitedMemoryBroydenSolver

source
DeepEquilibriumNetworks.LimitedMemoryBroydenSolver — Type
LimitedMemoryBroydenSolver()

Limited Memory Broyden Solver (Shaojie Bai, Vladlen Koltun, J. Zico Kolter (2020)) for solving Discrete DEQs.

See also: BroydenSolver
